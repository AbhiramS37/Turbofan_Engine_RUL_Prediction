# 🔧 Turbofan Engine Remaining Useful Life (RUL) Prediction

## 📌 Overview

This project predicts the Remaining Useful Life (RUL) of turbofan engines using machine learning on NASA CMAPSS dataset.
The system analyzes time-series sensor data and estimates how many cycles remain before engine failure.

---
## 🚀 Features
* RUL prediction using Random Forest, XGBoost, LightGBM
  
* Advanced feature engineering:
  
  * Rolling mean & standard deviation
  * Exponential weighted moving average (EWM)
  * Trend (slope) features
  * Cumulative drift (min/max)
  * Low-variance sensor removal

* Engine-level train/test split (prevents data leakage)
* Interactive Streamlit dashboard
* Health classification:

  * 🔴 Critical (RUL < 30)
  * 🟡 Warning (30–60)
  * 🟢 Healthy (≥ 60)

---

## 📊 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* XGBoost / LightGBM / RandomForest
* Streamlit

---

## 🧠 Model Approach

* RUL calculated as:
  RUL = max_cycle - current_cycle
* Capped at 125 cycles 
* RMSE used as evaluation metric

## ▶️ How to Run

### 1. Install dependencies

```
pip install -r requirements.txt
```

### 2. Preprocess data

```
python app/preprocess.py
```

### 3. Train model

```
python app/train.py
```

### 4. Run dashboard

```
streamlit run app/app.py
```
